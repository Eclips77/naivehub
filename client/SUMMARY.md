## 🚀 **איך להפעיל:**

### 1️⃣ **התקנה:**
```bash
cd client
pip install -r requirements.txt
```

### 2️⃣ **הפעלה:**
```bash
streamlit run naivehub_client.py
```

### 3️⃣ **גישה:**
